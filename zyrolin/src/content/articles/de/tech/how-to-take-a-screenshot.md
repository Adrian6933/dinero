Einen Screenshot zu machen ist eine der nützlichsten Sachen auf jedem Gerät. So geht's auf jeder großen Plattform.

## Windows

- **Ganzer Bildschirm:** Drücke **Druck** (kopiert in die Zwischenablage) oder **Windows + Druck** (speichert eine Datei).
- **Ausgewählter Bereich:** Drücke **Windows + Umschalt + S**, um das Snipping Tool zu öffnen.

## Mac

- **Ganzer Bildschirm:** **Cmd + Umschalt + 3**
- **Ausgewählter Bereich:** **Cmd + Umschalt + 4**, dann ziehen.

## iPhone

- Face-ID-Modelle: **Seitentaste + Lauter** drücken.
- Modelle mit Home-Taste: **Home + Seitentaste** drücken.

## Android

Auf den meisten Handys: **Power + Leiser** gleichzeitig drücken. Manche Handys unterstützen auch Handflächen-Wischgesten.

## Deinen Screenshot bearbeiten

Nach der Aufnahme kannst du im eingeblendeten Editor oder in jedem kostenlosen Bildbearbeiter zuschneiden, hervorheben oder kommentieren.

## Häufige Fragen

**Wo werden Screenshots gespeichert?**
Unter Windows im Ordner Bilder → Screenshots. Auf dem Mac standardmäßig auf dem Schreibtisch.

**Kann ich eine ganze Webseite als Screenshot aufnehmen?**
Ja – die meisten Browser haben eine Option „ganze Seite erfassen" in den Entwicklertools oder per Erweiterung.

## Fazit

Lerne das Kürzel für dein Hauptgerät, und Screenshots werden zur Selbstverständlichkeit.
