Los controladores son el software que permite a Windows comunicarse con tu hardware. Mantenerlos actualizados corrige errores, mejora el rendimiento y es esencial para jugar.

## Paso 1: Actualiza con Windows Update

1. Abre **Configuración → Windows Update**.
2. Haz clic en **Buscar actualizaciones**.
3. Abre **Opciones avanzadas → Actualizaciones opcionales** para encontrar actualizaciones de controladores.

Es el método más seguro para la mayoría de controladores.

## Paso 2: Actualiza los controladores gráficos

Los controladores de la tarjeta gráfica (GPU) se actualizan con más frecuencia, sobre todo para juegos. Descárgalos directamente desde la app oficial del fabricante de tu GPU, no de webs cualquiera.

> **Aviso:** Nunca descargues controladores de webs no oficiales: son una fuente común de malware.

## Paso 3: Crea primero un punto de restauración

Antes de actualizaciones de controladores importantes, crea un punto de Restaurar sistema para poder revertir si algo se rompe.

## Preguntas frecuentes

**¿Debería usar programas que actualizan controladores?**
Ten cuidado: muchos son bloatware. Las fuentes oficiales y Windows Update son más seguros.

**Mi juego se colgó tras una actualización de controlador. ¿Qué hago?**
Revierte el controlador en el Administrador de dispositivos, o restaura al punto anterior.

## Conclusión

Para la mayoría de controladores, Windows Update basta. Para jugar, actualiza el controlador de la GPU desde la app oficial y crea siempre antes un punto de restauración.
