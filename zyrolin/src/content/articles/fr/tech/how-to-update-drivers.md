Les pilotes sont le logiciel qui permet à Windows de communiquer avec votre matériel. Les garder à jour corrige des bugs, améliore les performances et est essentiel pour le jeu.

## Étape 1 : Mettre à jour via Windows Update

1. Ouvrez **Paramètres → Windows Update**.
2. Cliquez sur **Rechercher des mises à jour**.
3. Ouvrez **Options avancées → Mises à jour facultatives** pour trouver les mises à jour de pilotes.

C'est la méthode la plus sûre pour la plupart des pilotes.

## Étape 2 : Mettre à jour les pilotes graphiques

Les pilotes graphiques (GPU) se mettent à jour le plus souvent, surtout pour le jeu. Téléchargez-les directement depuis l'appli officielle du fabricant de votre GPU, pas depuis des sites quelconques.

> **Avertissement :** ne téléchargez jamais de pilotes depuis des sites non officiels : ce sont une source fréquente de malwares.

## Étape 3 : Créez d'abord un point de restauration

Avant des mises à jour de pilotes importantes, créez un point de restauration système pour pouvoir revenir en arrière si quelque chose casse.

## Questions fréquentes

**Dois-je utiliser des logiciels de mise à jour de pilotes ?**
Méfiance : beaucoup sont des bloatwares. Les sources officielles et Windows Update sont plus sûrs.

**Mon jeu a planté après une mise à jour de pilote. Que faire ?**
Revenez au pilote précédent dans le Gestionnaire de périphériques, ou restaurez votre point antérieur.

## Conclusion

Pour la plupart des pilotes, Windows Update suffit. Pour le jeu, mettez à jour le pilote GPU depuis l'appli officielle et créez toujours un point de restauration d'abord.
